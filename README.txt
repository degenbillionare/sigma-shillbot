# Twitter $SIGMA Bot

This bot automatically interacts with tweets mentioning "$sigma" by favoriting, retweeting, and replying with a random sigma GIF.

## Setup

1. Install required packages:

2. Create a `.env` file in the project root with the following content:

TWITTER_USERNAME=your_twitter_username
TWITTER_EMAIL=your_twitter_email
TWITTER_PASSWORD=your_twitter_password
TENOR_API_KEY=your_tenor_api_key

Replace the placeholders with your actual credentials.

## Running the Bot

To start the bot, simply run:

python bot.py

The bot will continuously search for tweets containing "$sigma", interact with them, and reply with sigma GIFs.
